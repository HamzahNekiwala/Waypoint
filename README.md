# Waypoint: Your Central Travel Hub

[cite_start]**Waypoint** is a centralized travel planning application designed to turn a "dream trip" into a simple and functional schedule[cite: 572, 574]. [cite_start]Developed by team **Cache Me Outside**, this platform allows users to catalogue flights, hotels, and activities in one place rather than searching through fragmented emails and maps[cite: 5, 573].

## ✨ Key Features (MVP)
* [cite_start]**Persistent Authentication:** Users log in with a username to save their specific itinerary to a persistent JSON file[cite: 531, 547].
* [cite_start]**Dependent Search Logic:** A smart search system where flight choices automatically filter available hotel dates (and vice-versa) to ensure itinerary alignment[cite: 635, 672].
* [cite_start]**Internal Data Catalog:** A curated internal JSON database ensures 100% uptime and a predictable testing environment[cite: 586, 628].
* [cite_start]**Itinerary Management:** A centralized view that summarizes all booked items in one list[cite: 529, 591].
